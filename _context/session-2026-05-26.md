## 设计决策记录

| 决定 | 理由 |
|------|------|
| Design & Production 用独立 preview 文件先做再组装 | 模块化开发，避免反复改主文件 |
| ASU swag 用 e-commerce 风格（5件一排，米色底）| 突出产品设计本身，背景统一让产品成为主角 |
| Buick 用 accordion 而非滚动条 | 53 张全部可见，一眼传达"same template, 53 colorways"的核心故事 |
| 图片裁切至 y=1005 保留 "Happy Chinese New Year!" | 完整呈现设计意图，移除二维码/头像等非设计信息 |
| tote + tshirt 从用户提供的官方宣传图裁出 | 质量更高，背景已是标准米色，无需额外处理 |
